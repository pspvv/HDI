About HDI Predictor

Understanding global human development through machine learning

What is HDI?

The Human Development Index (HDI) is a composite measure of life expectancy, education, and per capita income — ranking countries into four development tiers.

This project uses a Linear Regression model trained on real-world HDI data to deliver instant predictions with interactive charts, confidence scores, and batch CSV processing.
